# eventosti_ios
Descrição:
----------
App que exibe eventos de TI.

O projeto roda a partir do iOS 15.0, usa a arquitetura MVVVM e Persistência de dados off-line.

Dependências:
-------------
O App usa as seguintes dependências:

* Alamofire
* MaterialComponents/Buttons
* ObjectMapper
* Sheeeeeeeeet
* XCTest
